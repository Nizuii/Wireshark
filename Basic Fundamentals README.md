# What is a Wire Shark ?

- Wireshark is a network protocol analyzer also called packet sniffer.
- Wireshark Let's us see what is happening on a network at the packet level that includes every request, response, error, and handshake.
- It captures raw data into human readable form.

## Why IS Wire Shark used ?

- When something doesn't work but logs look fine:

  - Slow Internet
  - Packet loss
  - DNS Failures
  - TCP Handshake issues
  - Application timeouts
 
  Wire shark shows exactly where things break.

- In cybersecurity wireshark helps to:

  - Detect suspicious traffic.
  - Analyze malware communication.
  - Spot unencrypted credentials.
  - Inspect MITM attacks.
  - Analyze ARP spoofing, DNS poisoning.
